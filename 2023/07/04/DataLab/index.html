二、实验内容

1、bitXor

功能：实现x^y（异或）
运算符种类限制：、&
运算符个数限制：14
代码实现：
int bitXor(int x, int y)
{
  return ~(x&y)&(x&y);
}

2、tmin

功能：返回最小的补码整数
运算符种类限制：! ~ & ^ | + << >>
运算符个数限制：4
分析：最小补码整数为100000…0000
代码实现：
int tmin(void)
{
  return 1<<31;
}
